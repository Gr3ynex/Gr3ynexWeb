Website ini adalah template siap pakai untuk personal branding. 
Anda bisa menggunakannya untuk keperluan portofolio, CV online, atau profile profesional.

## Fitur
- Desain UI modern & minimalis
- Responsive (mobile & desktop)
- Struktur bersih dan mudah diedit
- Tanpa framework (HTML + CSS murni)
- Cocok untuk pemula maupun desainer web

## Cara Menggunakan
1. Ekstrak folder project.
2. Buka file `index.html` untuk melihat atau mengedit konten.
3. Semua style ada di `style.css`.
4. Untuk mengganti foto profil:
   - Masukkan gambar Anda ke folder `assets/`
   - Ganti path di bagian About:  
     `<img src="assets/profile.jpg">`

## Cara Upload ke Hosting
Jika pembeli ingin online-kan website:
1. Beli hosting + domain.
2. Masuk ke File Manager / FTP.
3. Upload semua file (index.html, style.css, assets/).
4. Website langsung tampil.

## Lisensi
Bebas digunakan untuk personal atau komersial.
Tidak boleh dijual ulang kecuali dalam bentuk modifikasi.
